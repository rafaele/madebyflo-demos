# Folio — Personal Portfolio Template

A warm, elegant single-page portfolio template for freelance designers and consultants. Built with vanilla HTML, CSS, and JavaScript — no dependencies, no build step.

## Sections
- **Hero** — Introduction with photo and experience badge
- **About** — Bio, stats, skill tags
- **Projects** — 4-project grid with categories
- **Services** — Brand Identity, Web Design, Strategy & Consulting
- **Testimonials** — Client quotes with avatars
- **Contact** — Form with name, email, project type, message
- **Footer** — Social links

## Features
- Light mode default with warm gold accents
- Fully responsive
- Scroll reveal animations
- Contact form with select dropdown
- Stats counter section
- Single-page layout (no routing needed)

## Design
- **Fonts:** DM Serif Display (headings) + Inter (body)
- **Accent:** Warm gold/orange
- **Aesthetic:** Clean, professional, approachable
