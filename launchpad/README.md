# Launchpad — SaaS Landing Page Template

A modern, dark SaaS landing page template with gradient accents. Built with vanilla HTML, CSS, and JavaScript — no dependencies, no build step.

## Sections
- **Hero** — Badge, headline, CTAs, product screenshot
- **Logos** — Trusted by company logos
- **Features** — 6-card grid (Analytics, AI, Collaboration, Security, Notifications, API)
- **Pricing** — Tiered pricing cards
- **Testimonials** — Customer quotes
- **CTA** — Final conversion section
- **Footer** — Product, Company, Legal links

## Features
- Dark mode with purple/blue gradient accents
- Fully responsive single-page layout
- Feature cards with icons
- Pricing table with highlighted plan
- Navigation with CTA button

## Design
- **Fonts:** Inter
- **Accent:** Purple-blue gradient
- **Aesthetic:** Modern SaaS, dark and polished
