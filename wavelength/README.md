# Wavelength — Music Artist Template

A dark-first, elegant website template for musicians, bands, and artists. Built with vanilla HTML, CSS, and JavaScript — no dependencies, no build step.

## Pages
- **Home** — Hero, latest release with tracklist, video grid, tour preview, about preview, newsletter
- **Music** — Full discography grid + featured album detail with streaming links
- **Tour** — Tour dates organized by leg/season with ticket links
- **About** — Bio, timeline, press quotes
- **Contact** — Contact form + email addresses by department

## Features
- Dark mode default (music industry aesthetic), light mode toggle
- Responsive down to 320px
- Scroll reveal animations
- Hide-on-scroll navigation
- Mobile hamburger menu
- Fluid typography (clamp-based)
- Gradient placeholders for artwork (ready for real images)
- Streaming platform badges
- Tour date cards with sold-out states
- Newsletter signup
- Press quotes section

## Design
- **Fonts:** Instrument Serif (headings) + Space Grotesk (body)
- **Accent:** Warm gold (#E8C36A)
- **Aesthetic:** Moody, intimate, editorial — not flashy
